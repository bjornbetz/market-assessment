# National Market Functionality Index

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/bjornbetz/market-assessment.git
   ```
2. Navigate to the project directory:
   ```bash
   cd market-assessment
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## How the Scraper Works
- The scraper is designed to collect data from various national market sources.
- It operates by sending requests to designated URLs, parsing the HTML content, and extracting relevant information.
- The data is then stored in a structured format for analysis.

## Deploying as a GitHub Pages Website
1. Ensure that your project is in a branch named `gh-pages` or has a `docs` folder in the main branch.
2. Go to the repository settings and locate the GitHub Pages section.
3. Choose the source for GitHub Pages (either `master` or `gh-pages` branch).
4. Visit `https://<username>.github.io/market-assessment` to view the deployed website.
